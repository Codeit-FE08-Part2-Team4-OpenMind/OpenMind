showToast(message, optionsopt)

Description:
  • 사용자 정의 스타일과 위치를 사용하여 토스트 메시지를 표시합니다.
Source:
  • components/@shared/Toast.jsx, line 32
Example
showToast('토스트 메시지가 표시되었습니다!', {
  duration: 5000,
  style: {
    background: 'blue',
    color: 'white',
    borderRadius: '12px',
    padding: '20px',
  },
  position: 'bottom-right'
});
Parameters:
Name Type Attributes Description
message string 표시할 메시지입니다.
options Object <optional>
 토스트 메시지의 옵션입니다.
Properties
Name Type Attributes Default Description
duration number <optional>
 4000 메시지가 표시되는 시간 (밀리초 단위). 기본값은 4000입니다.
style Object <optional>
 {} 토스트 메시지의 스타일을 사용자 정의합니다. 기본값은 빈 객체입니다.
position string <optional>
 'top-center' 토스트 메시지의 위치를 설정합니다. 기본값은 'top-center'입니다. 가능한 값은 'top-left', 'top-center', 'top-right', 'bottom-left', 'bottom-center', 'bottom-right'입니다.
